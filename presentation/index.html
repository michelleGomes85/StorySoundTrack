Transforme histórias
em emoções sonoras

Com o StorySoundTrack, você descobre a trilha sonora ideal para o seu livro. Nosso sistema analisa o clima geral da obra e sugere músicas que combinam perfeitamente com a atmosfera da narrativa.

Conheça o Repositório

Como Funciona

search

1. Escolha o Livro

Informe o título do livro e o autor (opcional). Seja o mais específico possível para encontrar o livro correto

sentiment_very_satisfied

2. Análise de Sentimentos

A IA do Gemini analisa o conteúdo do livro coletado via Google Books e identifica os sentimentos principais.

queue_music

3. Playlist Personalizada

Receba uma playlist com músicas que refletem o clima do livro, com links para o Spotify e uma explicação sobre cada escolha.

Tecnologias que Usamos

Google Books API

Buscamos informações sobre o livro, como sinopse e capa, para criar a experiência perfeita.

Gemini API

Nossa IA analisa o livro e identifica os sentimentos principais para criar a playlist ideal.

Spotify API

Integramos músicas reais do Spotify para que você possa curtir a playlist diretamente na plataforma.
